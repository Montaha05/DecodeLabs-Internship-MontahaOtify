# Iris Flower Classifiaction Using K-Nearest Neighbors(KNN)

## Overview
This project was developed as part of the DecodeLabs AI internship Program

The objective is to build a supervised machine leaning classification model capable of predicting Iris flower species using flower measurments.

The project follows a complete ML workflow, including:
* Data Loading
* Exploratory Data Analysis (EDA)
* Data Preprocessing
* Feature Scaling
* Train-Test Split
* K-Nearest Neighbors (KNN)
* Model Evaluation

---

## Dataset

The project uses the Iris dataset, one of the most widely used dataset in machine leaning.

**Features:**
* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

**Taget Classes:**
* Setosa
* Versicolor
* Virginica
