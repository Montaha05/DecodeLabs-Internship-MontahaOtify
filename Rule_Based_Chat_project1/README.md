# Rule-Based AI Chatbot
## Overview
This project is a simple rule-based chatbot developed as part of the DecodeLabs AI internship program.

The Chatbot uses predefined intents and pattern matching to simulate basic conversations without using machine learning or external AI libraries.

The purpose of this project is to demonstrate fundamental AI concepts such as decision-making logic, control flow, user interaction, and intent recognition.

## Features
* Continuous conversation loop
* Input sanitization using ```strip()``` and ```lower()```
* Intent-based pattern matching
* Multiple predefined conversation intents
* Fallback response for unknown inputs
* Exit command handling

## Technologies Used 
* Python
* Dictionaries
* Loops
* Conditional Statements
* String Processing

## Supported Intents
* Greetings
* Asking the Chatbot's name
* Asking what the chatbot can do
* Asking how the chatbot is doing
* Jokes
* Help
* Exit commands

## Example Conversation

You: hello

Chatbot: Hi there! How can I help you today?

You: tell me a joke

Chatbot: Why don't programmers like nature? Because it has too many bugs!

You: exit

Chatbot: Goodbye!
