# DecodeLabs AI Internship Portfolio

> A collection of Artificial Intelligence projects completed during my internship at DecodeLabs, covering fundamental AI concepts from rule-based systems to machine learning, recommendation engines, and computer vision.

## Overview

This repository showcases the projects developed throughout my Artificial Intelligence Internship at DecodeLabs. The internship was structured as a progressive learning journey, starting with logic-based AI systems and advancing toward machine learning, recommendation systems, and Optical Character Recognition (OCR).

Each project focuses on a different area of Artificial Intelligence and demonstrates practical implementation using Python and industry-standard libraries.

## Internship Learning Path

### Project 1 — Rule-Based AI Chatbot

**Focus:** AI Fundamentals & Decision Logic

Developed a chatbot capable of responding to user inputs using predefined intents and rule-based pattern matching. The project emphasizes control flow, decision-making logic, and conversational interaction without machine learning.

**Key Concepts**

* Rule-based AI
* Intent recognition
* Pattern matching
* Conditional logic
* User interaction design

**Technologies**

* Python
* Dictionaries
* Loops
* Conditional Statements

---

### Project 2 — Iris Flower Classification Using KNN

**Focus:** Supervised Machine Learning

Built a classification model using the famous Iris dataset to predict flower species based on physical measurements. The project follows a complete machine learning workflow including preprocessing, training, and evaluation.

**Key Concepts**

* Supervised Learning
* Classification
* Feature Scaling
* Train-Test Split
* Model Evaluation

**Technologies**

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

**Results**

* Accuracy: 93.33%
* Weighted F1 Score: 93.27%
* Best Observed Accuracy: 96.67%

---

### Project 3 — AI Tech Stack Recommender

**Focus:** Recommendation Systems

Developed a content-based recommendation engine that suggests technology career paths based on a user's existing skills. The system calculates similarity scores between user profiles and predefined role requirements.

**Key Concepts**

* Recommendation Systems
* Similarity Scoring
* Skill Matching
* User Profiling
* Flask Web Applications

**Technologies**

* Python
* Flask
* Pandas

---

### Project 4 — OCR Recognition System

**Focus:** Computer Vision & Text Recognition

Built an Optical Character Recognition (OCR) system capable of extracting text from images using Tesseract OCR and OpenCV. The project includes image preprocessing, confidence evaluation, batch processing, and an interactive Gradio interface.

**Key Concepts**

* Optical Character Recognition (OCR)
* Image Processing
* Confidence Analysis
* Computer Vision
* Human-Computer Interaction

**Technologies**

* Python
* OpenCV
* Tesseract OCR
* Pandas
* NumPy
* Gradio
